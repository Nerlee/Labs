Возможности

Добавление уроков - создавайте записи с датой, временем и названием
Сохранение - экспортируйте данные в текстовый файл
Загрузка - импортируйте ранее сохранённые расписания
Удаление - удаляйте ненужные записи из списка

Как использовать

1. Добавление урока

  1. Нажмите кнопку "Добавить"
  2. Выберите дату и время
  3. Введите название урока
  4. Нажмите OK

2. Сохранение

  1. Нажмите "Сохранить"
  2. Выберите место для файла
  3. Готово!

3. Открытие файла

  1. Нажмите "Открыть"
  2. Выберите файл .txt
  3. Данные загрузятся в таблицу
Внимание: При открытии нового файла старые данные будут удалены!

Формат файла

Данные сохраняются в текстовом файле в кодировке UTF-8:

15.09.2025 10:30:00 "Математика"
16.09.2025 14:00:00 "Физика"
17.09.2025 09:00:00 "История"

Формат: ДД.ММ.ГГГГ ЧЧ:ММ:СС "Название"
